# Утиліта: Планувальник покупок (Варіант №18)

Консольна утиліта на Python для автоматизації планування побутових покупок, контролю витрат та ведення списку необхідних товарів. Розроблено в межах навчальної практики.

## Функціонал програми
- Перегляд поточного списку покупок із підрахунком загальної суми до сплати.
- Додавання нових товарів із валідацією введених даних.
- Позначення товарів як куплених.
- Видалення товарів зі списку за їх ID.
- Автоматичне збереження даних у файл `shopping_list.json` між запусками програми.

## Структура проєкту
- `main.py` — основний скрипт програми та CLI-інтерфейс.
- `shopping_list.json` — файл збереження даних у форматі JSON (створюється автоматично).
- `test_shopping.py` — автоматизовані тести (модуль `unittest`).
- `.gitignore` — файл виключень для Git.

## Інструкція із запуску (з чистого аркуша)
1. Переконайтеся, що на комп'ютері встановлено **Python 3.x**.
2. Запустіть головний файл програми через термінал:
   ```bash
   python main.py
