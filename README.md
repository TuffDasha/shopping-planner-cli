# Програмний продукт: Планувальник покупок (Варіант №18)

Сучасний десктопний застосунок із графічним інтерфейсом користувача (GUI) на базі бібліотеки `tkinter` для автоматизації планування побутових покупок, розподілу товарів за категоріями, контролю фінансових витрат та ведення архіву.

---

## Функціонал програми
- **Графічний інтерфейс (GUI):** зручна таблиця `Treeview` для відображення товарів з автоматичним скролбаром.
- **Динамічна статистика:** миттєвий підрахунок кількості показаних товарів, загальної вартості списку та суми «До оплати» за некуплені позиції.
- **Інтерактивне керування статусом:** можливість в один клік позначити товар як «Куплений» або видалити його за допомогою кнопок на панелі дій.
- **Розумна фільтрація:** роздільний перегляд «Активного списку», «Архіву» куплених товарів, а також фільтрація за категоріями через випадаючий список `Combobox`.
- **Надійна валідація:** захист від некоректного введення даних (перехоплення `ValueError` для ціни та кількості з виведенням графічних вікон `messagebox`).
- **Автоматична серіалізація:** збереження та завантаження даних у файл `shopping_list.json` у кодуванні UTF-8.

---

## Структура проєкту
Репозиторій містить повний набір файлів, необхідних для автономного запуску програми:
- `main.py` — головний модуль програми: логіка обробки даних та реалізація графічного інтерфейсу (GUI).
- `test_shopping.py` — автоматизовані модульні тести (`unittest`) для перевірки обчислювальних функцій бізнес-логіки.
- `shopping_list.json` — локальна база даних у форматі JSON (створюється автоматично при першому запуску програми або додаванні товару).
- `.gitignore` — конфігураційний файл для виключення системних файлів з відстеження Git.
- `README.md` — документація проєкту та інструкція користувача.

## Інструкція із запуску
### 1. Клонування репозиторію
Відкрийте термінал (або командний рядок) та виконайте команду:
git clone [https://github.com/TuffDasha/shopping-planner-cli.git](https://github.com/TuffDasha/shopping-planner-cli.git)
cd shopping-planner-cli 

### 2. Запуск програми
Переконайтеся, що на комп'ютері встановлено Python 3.x. Запустіть головний файл програми:
# Для Windows:
python main.py
# Для Linux / macOS:
python3 main.py

### 3. Запуск автоматизованих тестів
Щоб перевірити коректність роботи утиліти за допомогою юніт-тестів, виконайте:
# Для Windows:
python -m unittest test_shopping.py
# Для Linux / macOS:
python3 -m unittest test_shopping.py
